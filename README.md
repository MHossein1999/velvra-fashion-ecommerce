# VELVRA — Fashion E-Commerce Demo

A minimalist, editorial-style fashion e-commerce site built to showcase pixel-perfect frontend design, luxury branding, and responsive layout techniques. Designed for women’s fashion with a focus on emotional warmth, curated imagery, and elegant typography.

---

## 🧠 Design Goals

- Create an editorial hero layout that evokes luxury and rhythm
- Use soft gradients and curated imagery to balance minimalism with emotional resonance
- Build a responsive product grid that adapts seamlessly across devices
- Showcase frontend precision through typography, spacing, and hover interactions

---

## 🛠️ Tech Stack

- **Next.js** — for fast, scalable React-based architecture
- **Tailwind CSS** — for utility-first styling and layout control
- **TypeScript** — for type safety and maintainability
- **@fontsource** — for self-hosted font reliability
- **PowerShell & CMD** — for Windows-based project cleanup and lockfile management

---

## 🎨 Design Decisions

- **Font Pairing**: Cormorant for elegance, Inter for clarity
- **Hero Layout**: Two-line heading with adaptive breakpoints for editorial rhythm
- **Color Palette**: Soft neutrals and warm tones to evoke luxury
- **Image Curation**: Editorial-style product and category visuals selected for emotional impact

---

## 🧩 Features

- Responsive hero section with adaptive typography
- Category showcase with hover effects and grid layout
- Feature collection with curated product cards
- Footer with email subscribe CTA and clean UI
- Mobile-first design with layout refinements for small screens

---

## 🧪 Challenges Solved

- Resolved layout overflow and spacing issues for exact screen fit
- Adapted AI-generated code to match brand tone and design precision
- Switched from Google Fonts to local hosting for performance and reliability
- Troubleshot peer dependency conflicts with React 19 and cleaned lockfiles on Windows
